Creates an Openstack Instance object, from an already globally defined cloud name and label without registering it as a Jenkins agent.
This step has a third optional String parameter, named scope, reflecting the booted Server scope, with possible values from:
run
The booted Instance will be kept alive whilst the current run is kicking. This is the default mode if no scope is defined.
unlimited
The booted Instance will be kept alive forever. Mind the possible resources extarvation at the Openstack provider!
